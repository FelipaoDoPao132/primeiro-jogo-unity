# 🎮 Jogo de Plataforma 2D

Um jogo de plataforma bidimensional desenvolvido na Unity. O projeto está em fase de desenvolvimento (V0.0.2), contando atualmente com a primeira fase totalmente funcional e mecânicas base estruturadas.

## 🚀 Como Jogar

Para testar o jogo diretamente no seu computador:
1. Vá até a seção **[Releases](https://github.com)** do repositório.
2. Baixe o arquivo arquivo `.zip` da versão mais recente.
3. Extraia o conteúdo do arquivo em uma pasta no seu computador.
4. Execute o arquivo `.exe` do jogo.

## 🕹️ Controles

* **`A`** - Mover para a esquerda
* **`D`** - Mover para a direita
* **`Espaço (Spacebar)`** - Pular

## 🛠️ Recursos Implementados (V0.0.2)

* **Menu Principal:** Tela inicial de navegação.
* **Sistema de Moedas:** Sistema de coleta de itens com persistência e contagem.
* **GameManager:** Arquitetura para controle centralizado do estado e fluxo do jogo.
* **Transição de Fases:** Sistema de portal funcional que realiza a troca de cenas ao entrar em contato.
* **Ajuste de Colisões:** Correções de física aplicadas nas nuvens e plataformas para melhor fluidez do movimento.
* **Visual Integrado:** Adição de sprites para o jogador, plataformas e portais.

## 💻 Tecnologias Utilizadas

* **Engine:** Unity
* **Linguagem:** C#
